CSS border applied to div element : border: groove #000 2px works on Firefox, not Webkit.
CSS box-shadow applied : works everywhere ! 
                
                box-shadow: 0 10px 5px 0 #000, 10px 0 5px 0 blue,
                            0 -10px 5px 0 red , -10px 0 5px 0 gold;
The syntax : 
                box-shadow: none | <shadow> [ , <shadow> ]*
                <shadow> = inset? && [ <length>{2,4} && <color>? ]
Order of shadow if length = 4 and not inset : bottom, right, top, left
Length : offset-x (required), offset-y (required), blur-radius, spread-radius
More info on css3.info
Example with 3 shadows 
                box-shadow: 10px 10px #888, 
                           -10px -10px #000, 
                             0px 0px 5px 5px #cc6600
Example with inset and two colors 
                box-shadow: 10px 5px 5px #000 inset, 
                           -10px -5px #888 inset;
Example with inset and two colors 
                box-shadow: 0 1px 5px #000 inset, 
                            0 -1px #888 inset;
